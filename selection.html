Selection if-else-elif

Decision Structure
if

คำสั่ง if เป็นคำสั่งที่ใช้ควบคุมการทำงานของโปรแกรมที่เป็นพื้นฐานและง่ายที่สุด เราใช้คำสั่ง if เพื่อสร้างเงื่อนไขให้โปรแกรมทำงานตามที่เราต้องการเมื่อเงื่อนไขนั้นตรงกับที่เรากำหนด เช่น การตรวจสอบค่าในตัวแปรกับตัวดำเนินการประเภทต่างๆ นี่เป็นรูปแบบของการใช้งานคำสั่ง if ในภาษา Python.

จากรูปหากเงื่อนไขที่กำหนดเป็นจริง จะทำการดำเนินการตามขั้นตอนถัดไปดังภาพ

if-else

คำสั่ง if-else จะดำเนินการหนึ่งบล็อกของคำสั่งหาก เงื่อนไขเป็นจริง หรือบล็อกอื่นหากเงื่อนไขเป็นเท็จ

จากรูปหากเงื่อนไขเป็นจริงจะดำเนินการด้าน block a หาก เงื่อนไขเป็นเท็จจะดำเนินการด้าน block b
