STRING

String ในภาษา python
if

String เป็นลำดับของตัวอักษรหลายตัวเรียงต่อกัน ซึ่งในภาษา Python นั้นการที่จะประกาศ String ค่าของมันจะอยู่ในเครื่องหมาย Double quote หรือ Single quote เท่านั้น มาดูตัวอย่างการประกาศตัวแปรของ String

STRING FUNTIONS LEN()

len(obj) เป็น built-in function ใช้หาจำนวนที่อยู่ในวัตถุนั้น โดยวัตถุนั้นจะเป็น string, tuple, list หรือ range ก็ได้ หรือจะเป็นกลุ่มของข้อมูล เช่น dictionary หรือ set เป็นต้น
ถ้าใช้ len ฟังก์ชั่นกับ string มันจะบอกเราว่ามีตัวอักษรกี่ตัวใน string นั้น หากใช้กับ list มันจะบอกว่าใน list นี้มีสมาชิกอยู่กี่ตัว

จากผลลัพธ์ จะสังเกตุได้ว่า มี string ทั้งหมด 9 ตัว

Basic String method

1.upper() : Return a copy of the string in uppercase.


2.lower() : Returns a copy of the string in lowercase.


3. format() : Returns a formatted version of the string.
