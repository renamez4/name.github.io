REPETITION STRUCTURE.

โครงสร้างแบบวนซ้ำ, โครงสร้างการทำงานที่มีการทำงานในบางคำสั่งหรือบางชุดของคำสั่งซ้ำกันมากกว่าหนึ่งรอบขึ้นไป.

CONDITION-CONTROLLED LOOP.

คำสั่งวนลูป เป็นคำสั่งที่ใช้ควบคุมการทำงานของโปรแกรมแบบวนซ้ำ การทำงานเดิมๆตามเงื่อนไขที่ดำหนด เช่น การวนซ้ำการทำงานเดิมเป็นจำนวน 10 รอบ ทำงานซ้ำๆจนกว่าเงื่อนไขจะเป็นเท็จ จะมีชุดคำสั่งให้ทำงานเป็นรอบอยู่ 2 รูปแบบด้วยกัน คือ
1.การทำงานเป็นรอบด้วย While
1.การทำงานเป็นรอบด้วย For

WHILE

ประโยคคำสั่ง while มีรูปแบบการเขียนดังนี้ คือ รูปแบบ :
while (expression)
{
statement;
}
โดยที่ : expression หมายถึง นิพจน์ที่นำมาใช้ตรวจสอบเงื่อนไข
statement หมายถึง ประโยคคำสั่งที่งานซ้ำ
ลักษณะการทำงานของ LOOP WHILE
1. ชุดคำสั่งภายในลูป while จะทำงานเมื่อเป็นจริง (True)
2.การออกจากลูปเมื่อคำสั่งเงื่อนไขเป็นเท็จ (false)
3.เงื่อนไขที่นำมาตรวจสอบ สามารถใช้ตัวดำเนินการเปรียบเทียบ หรือตำดำเนินการทางตรรกะได้ เช่น while (i = 10)

FOR

ลูป For ใน Python คืออะไร ลูป for (ฟอร์) เป็นลูปชนิดหนึ่งในภาษาไพธอน หลักการทำงานก็คล้ายๆภาษาอื่นๆ ตัวอย่างการทำงาน ของ for loop เช่น

จะเห็นได้ว่า i มีค่าตั้งแต่ 0-3
