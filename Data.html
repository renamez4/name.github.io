ตัวแปลภาษา

1.Compiler
2.Interpreter

Compiler คือ การใช้คอมไพเลอร์ จะต้องเขียนโปรแกรมเป็นไฟล์ Source Code ให้เสร็จซะก่อน แล้วใช้คอมไพเลอร์ทำการคอมไพล์ไฟล์ ให้ออกมาเป็น Execution File

Interpreter จะอ่านไฟล์แล้วแปลคำสั่งเหมือนคอมไพเลอร์ แต่จะไม่แปลพร้อมกันทั้งไฟล์ อินเทอร์พรีเตอร์จะอ่าน Source Code ทีละบรรทัด (หรือทีละคำสั่ง) แล้วแปลงเป็น Machine Codeแล้วทำการ Execute คำสั่งนั้นตอนนั้นเลย

Input
